# KlikIDM_DS

KlikIDM_DS is an iOS SDK that contains components and animations that can be used to help developers design user interfaces based on the Klik Indomaret Design System.

## Dependencies
- Support iOS version from 12.0
- Support Swift 5
- Support UIKit only
